# 👾 DIGITAL DEFENDER - Standalone Game

This folder contains the complete, ready-to-deploy **DIGITAL DEFENDER** arcade game.

## 📁 Files

- `index.html` — Main game page
- `game.js` — Complete game logic and mechanics

## 🚀 How to Deploy to Vercel

1. **Create a Vercel account** at vercel.com
2. **Upload this folder** to Vercel (drag & drop or Git integration)
3. **Vercel automatically serves** `index.html` as the entry point
4. **Your game goes live** instantly!

## 🎮 Features

- Arcade action gameplay with wave-based difficulty
- 3 lives system
- High score persistence (saved in browser localStorage)
- Smooth keyboard + mouse controls
- Progressive wave scaling
- Two enemy types: basic enemies and bosses

## 📊 Game Stats

- **Canvas**: 800x600 pixels
- **Lives**: 3
- **Waves**: Infinite progression
- **Score**: Persistent high score tracking

## 🎯 Controls

- **WASD** or **Arrow Keys** — Move your ship
- **Mouse** — Aim and shoot
- **Click Restart Button** — Play again after game over

Enjoy! 🎮
